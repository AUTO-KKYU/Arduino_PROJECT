### Configuration
- 최소 3층 이상 엘레베이터 구현
- 엘레베이터의 현재 위치를 나타내는 LED 구성
     - 1층,2층,3층 : RED
     - 층간 사이 2개씩 총 4개 : YELLOW
- 각 층마다 호출 버튼 있어야 함
- 각 층마다 호출 상태를 나타내는 LED 있어야 함
